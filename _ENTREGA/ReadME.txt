Nombre del Proyecto: Multiplicación de Matrices en C

Descripción: Este proyecto implementa la multiplicación de dos matrices cuadradas de tamaño n x n en el lenguaje de programación C. El programa permite al usuario ingresar los valores de las matrices A y B y calcula la matriz resultante C, que es el producto de A y B. La implementación también incluye una función para imprimir matrices en la consola en formato bidimensional.

Características:

Interactividad: Los valores de las matrices A y B son ingresados por el usuario, lo que permite trabajar con diferentes conjuntos de datos sin necesidad de modificar el código fuente.
Flexibilidad: El tamaño de las matrices n x n es definido por el usuario en tiempo de ejecución.
Modularidad: El código está dividido en funciones claras y separadas para la impresión de matrices y para la multiplicación de las mismas.
Funciones Principales:

imprimirMatriz: Imprime una matriz unidimensional como una matriz bidimensional en la consola.
multiplicarMatrices: Calcula el producto de dos matrices cuadradas de tamaño n x n y almacena el resultado en una tercera matriz.
Uso:

El usuario ingresa el tamaño de las matrices n x n.
El usuario ingresa los valores para cada elemento de las matrices A y B.
El programa multiplica las matrices A y B, almacenando el resultado en la matriz C.
Las matrices A, B y C son impresas en la consola.
Ejecución:
Este programa puede compilarse y ejecutarse en cualquier entorno que soporte el compilador de C. Para compilar, utiliza el siguiente comando:


gcc -o multiplicacion_matrices  Tarea_Multiplicacion_Matrices_con_3_For_anidados.c
Para ejecutar el programa:

./multiplicacion_matrices
